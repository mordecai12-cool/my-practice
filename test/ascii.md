What you are building

ASCII Art is a Go program that receives a string and prints a large graphic representation of that string using ASCII characters.

The program reads pre-made banner files that define how each printable character should look.
Supported input

Your program must handle numbers, letters, spaces, special characters, and literal `\n` sequences in the input string.

An empty string should produce no visible banner output. A lone `\n` should print a newline.
Banner files

Three banner styles are provided: shadow, standard, and thinkertoy.

Standard banner: https://raw.githubusercontent.com/01-edu/public/master/subjects/ascii-art/standard.txt

Shadow banner: https://raw.githubusercontent.com/01-edu/public/master/subjects/ascii-art/shadow.txt

Thinkertoy banner: https://raw.githubusercontent.com/01-edu/public/master/subjects/ascii-art/thinkertoy.txt
